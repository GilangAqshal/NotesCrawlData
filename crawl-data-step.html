Bagian 2 — Proses Scraping
🌾
Eksekusi Tweet Harvest
tweet-harvest@2.6.1 · search · latest tab · csv output
06
Eksekusi Scraping Twitter
tweet-harvest · keyword · limit · csv
Python
Jalankan proses scraping dengan keyword "Piala Dunia" lang:id dan ambil 1000 tweet terbaru. Hasilnya disimpan ke file CSV.
python 
# Pengaturan parameter pencarian
filename       = "piala_dunia_2026.csv"
search_keyword = '"Piala Dunia" lang:id'
limit          = 1000

# Menjalankan tweet-harvest
!npx -y tweet-harvest@2.6.1 \
  -o "{filename}" \
  -s "{search_keyword}" \
  --tab "LATEST" \
  -l {limit} \
  --token {twitter_auth_token}

10
Ekspor Hasil Akhir ke Format JSON
to_json · orient records · indent
Python
Ekspor data ke file JSON. Pilih Opsi 1 untuk ekspor semua kolom, atau Opsi 2 (aktifkan dengan menghapus #) untuk ekspor teks tweet saja.
python 
# Opsi 1: Seluruh kolom → JSON
df.to_json(
  '/content/tweets-data/piala_dunia_2026.json',
  orient='records',
  indent=4
)
